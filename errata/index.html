My Errata Page for Yaakov Chaikin's Coursera Web Development Class

Everything here reflects my own experiences with the course in January 2021, as well as my related thoughts and ideas. None of it is endorsed by Prof. Chaikin or anyone else. Prof. Chaikin has his own FAQ where a number of changes have been noted: Official Course FAQ

Week 1

GitHub has changed since the course was created, and the Official FAQ covers the key point -- just work in the "main" branch, not gh-pages.

When I got to the coding exercise in Module 2, I realized I had forgotten about everything about using Git and GitHub. So here are some references:

  • 'Hello World!' -- very basic introduction
  • GitHub Flow Guide -- understanding the process
  • Basic GitHub Handbook
  • Using GitHub Pages -- putting your website on GitHub

Week 2

In Lecture 25, the professor says that jQuery is required to use Bootstrap, but that is no longer true. According to the Bootstrap documentation, you no longer need jQuery to use Bootstrap.

That said, as the link above notes, if you still want to use jQuery, you can.
